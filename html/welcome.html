Welcome to Lau Wei Tang JS Backend Project

This backend project is deployed on Google App Engine

This is a simple application that stores anything you want

Ranging from your lucky numbers to your groceries, this is your go to app

Here are the APIs

GET / Gets this page

GET /store list all contents stored

GET /store/{index} get the content stored at the particular index (zero-based)

POST /store/{value} inserts the content at the back of the array

POST /store/{index}/{value} inserts the content at the specified index (zero-based)

PUT /store/{value} replaces the content at the back

PUT /store/{index}/{value} replaces the content at the specified index (zero-based)

DELETE /store/all deletes all content

DELETE /store deletes the last indexed content

DELETE /store/{index} deletes the content at the specified index
